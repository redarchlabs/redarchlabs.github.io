What you're licensing

Red Arch Knowledge Manager is a complete, self-hostable platform — open source under Apache 2.0, and built to integrate and move with your organization.

Built-in REST API

A versioned REST API at /api/v1 with org-scoped API keys, scopes, and rate limiting — automate records, reports, workflows, search, and knowledge from anything.

Import & export

Move an entire org as a portable JSON bundle — entities, forms, views, workflows, records, and documents — to back up, clone, or migrate between installs.

Change management

Promote configuration across dev, staging, and production with lineage-aware diffs and controlled releases — ship changes safely between environments.

Role-based security

Fine-grained, per-role access to every document, record, and view — so each person sees exactly what they should and nothing they shouldn't.

AI chat over your knowledge

Ask questions and get answers grounded in your own content, with passage-level citations back to the source document.

Self-hostable

Run it on your own infrastructure and own your data end to end. No lock-in — it's yours under Apache 2.0.
